# 商业模式: 从200万到1亿的路径
> 作者: CEO
> 日期: 2026-04-08
> 状态: 讨论稿，待董事会确认

## 核心命题

200万启动资金，18个月1亿营收。这意味着平均月增长需要极高。不能靠线性增长，必须找到指数级拐点。

## 收入模型: 三层结构

### Layer 1: 接入部署费 (一次性)
- 单个设备品牌/型号的数字底座接入收费
- 参考定价: 5-10万/品牌 (含首次接入调试+认证)
- Phase 1目标: 接入5+品牌, 部署费收入 25-50万
- 这层收入天花板低，主要价值是建立合作关系和数据入口

### Layer 2: 激活授权费 (按设备)
- 随设备出货的中枢接入授权
- 参考定价: 200-500元/台/年
- Phase 1目标: 若首批5家厂商年出货10万台设备, 渗透率10% = 1万台授权
- 收入预估: 20-50万/年 (Phase 1贡献有限)
- Phase 2-3爆发点: 出货量指数增长

### Layer 3: 订阅服务费 (SaaS)
- B2B机构SaaS订阅 (健康管理平台、干预引擎)
- 参考定价: 2-5万/机构/年
- Phase 1目标: 3-5家机构, 10-25万
- Phase 2-3: 30+机构 = 60-150万/年
- **这是唯一能支撑1亿目标的收入层**

## 1亿营收路径分析

### 现实检查: 18个月到1亿是否可行?

保守测算:
- 接入部署: 50万
- 激活授权: 100万 (需万台级设备出货)
- SaaS订阅: 150万 (需30+机构)
- Phase 1总计: ~300万

Phase 2规模化 (月12-18):
- 接入部署: 200万 (20+品牌)
- 激活授权: 500万 (5万+设备)
- SaaS: 500万 (50+机构)
- Phase 2总计: ~1200万

**18个月累计: ~1500万**

### 结论: 18个月1亿在B2B模式下不现实

需要调整预期或引入其他收入来源:

**路径A: 放宽时间线到36个月**
- 36个月通过B2B+数据服务+2G可以做到5000万-1亿

**路径B: 引入资本加速**
- 在Phase 1验证后(6-12月)融资, 用资本换取市场速度
- 目标融资: 1000-3000万
- 估值锚点: Phase 1的设备接入数、用户数据资产、标杆客户

**路径C: 加入2C收入**
- B2B2C模式下, C端订阅费(健康管理会员)
- 定价: 99-299元/年
- 10万活跃用户 = 1000-3000万/年
- 这是1亿目标的核心贡献者

## 推荐路径: B + C 组合

### 阶段规划

| 时间 | 关键动作 | 收入目标 | 关键指标 |
|------|---------|---------|---------|
| 月1-3 | 样板机型验证, 首批2-3家B2B客户签约 | 30万 | 3+品牌接入, 2+机构签约 |
| 月4-6 | 数据飞轮启动, 验证干预效果 | 80万 | 1000+用户数据, 干预场景闭环 |
| 月7-9 | 多厂商复制, B2B规模化 | 200万 | 10+品牌, 5+机构 |
| 月10-12 | 启动融资(目标1000-3000万) | 400万 | 10+机构, 数据飞轮验证 |
| 月13-15 | 融资到账, B2B2C启动 | 800万 | C端产品上线 |
| 月16-18 | C端增长 + 2G探索 | 1500万 | 5万+MAU, 1+2G项目 |

**18个月累计: ~3000万**

### 要到1亿, 需要追加:
- 月19-24: C端规模化 + 2G项目签约 (4000万)
- 月25-30: PaaS平台 + 生态收入 (3000万)
- **30个月达1亿**

## 健身房ROI模型验证

- 单店设备投入 E = 50万 (10台设备)
