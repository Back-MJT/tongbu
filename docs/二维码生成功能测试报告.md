## 使用说明

### 管理系统操作步骤
1. 访问 http://localhost:80
2. 登录管理系统
3. 进入"器械管理"页面
4. 在器械列表中找到需要生成二维码的器械
5. 点击该器械行的"二维码"按钮
6. 在二维码页面可以：
   - 查看器械信息和二维码
   - 点击"复制"按钮复制二维码内容
   - 点击"下载二维码"保存图片
   - 点击"打印二维码"打印标签

### 小程序使用步骤
1. 打开昕动智能小程序
2. 进入"器械训练"页面
3. 点击"扫码开始训练"
4. 扫描器械上的二维码
5. 系统自动识别器械并连接传感器
6. 开始训练

## 注意事项

1. **二维码打印建议**：
   - 建议打印尺寸：5cm x 5cm 或更大
   - 使用防水标签纸
   - 贴在器械显眼位置

2. **蓝牙名称配置**：
   - 确保每个器械都配置了正确的蓝牙名称
   - 蓝牙名称应与实际传感器的广播名称一致

3. **测试建议**：
   - 先在管理系统中生成二维码
   - 打印或在屏幕上显示二维码
   - 使用小程序扫描测试
   - 验证能否正确识别和连接
