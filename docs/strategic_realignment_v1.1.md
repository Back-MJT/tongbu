## 1. 董事会决议摘要

董事会批准了 HealthHub 产品技术规划 v1.1。核心变化：

- **定位**: 中国健身器械行业的**数字底座 + 个性化干预引擎**（Board修订：原"数字神经系统"比喻不完整，已改为功能性描述）
- **目标市场**: 山东省德州市宁津县 — **28家规上企业为核心**，兼顾创新派中小厂家（Board修订：400+家中规上仅28家，聚焦核心）
- **启动资本**: ¥200万，18个月目标**¥1亿公司现金流入**（Board修订：原"产业总产值"理解有误，Board明确为现金流）
- **团队规模**: 3-5人精英团队
- **四层产品体系**: IMU硬件模组(第零层) → 设备数字化SDK(第一层) → AI训练引擎(第二层) → 数据价值(第三层)
- **执行策略**: 双轨并行 — 快轨（有屏设备SDK，3个月闭环）+ 慢轨（IMU模组开发，长期壁垒）
- **LLM引擎**: callLLM()抽象层支持多Provider热切换（Claude→DeepSeek→Qwen→本地部署）

## 4. 干预引擎(intervention-engine)处理方案

现有干预引擎包含715个通过的测试，代码质量高。处理方式：

- **保留**: 基础架构(Pydantic models, 配置系统, 数据管道)
- **重构**: 算法核心从循证规则引擎转向Claude API调用引擎
  - `evidence_registry.py` → 改为训练方案模板库
  - `health_profile_v2.py` → 改为用户运动画像
  - `sleep_intervention.py` / `heart_rate.py` → 降级为参考模块
  - `ab_testing.py` → 保留，用于System Prompt A/B测试
  - `feedback_loop.py` → 保留，用于用户反馈驱动的引擎优化
- **新建模块**:
  - `claude_engine.py` — callClaude()封装，含缓存、日志、成本追踪
  - `training_plan.py` — 个性化训练方案生成
  - `user_stage.py` — 用户阶段识别(初学/成长/停滞/进阶)
  - `achievement.py` — 成就徽章系统

## 8. Board补充 (XIN-98, 2026-04-14)

Board在v1.1执行过程中提出4项战略补充：

### 8.1 使用者画像优先

产品底层逻辑相通，但应用场景和人群不同。产品设计必须从使用者画像出发：

| 场景 | 核心人群 | 习惯特征 | 产品设计启示 |
|------|---------|---------|------------|
| 健身房 | 年轻人（20-35岁）| 追求效率、社交分享、数据驱动 | 组数计数、进步可视化、社交功能 |
| 老年康复馆 | 刚退休/退休几年（55-70岁）| 不甘日常、释放被束缚的心、追求人生感 | 情感激励、简单交互、成就感、人生第二舞台 |
| 家庭 | 妇女/家庭用户（25-50岁）| 碎片时间、健康焦虑、家庭共享 | 灵活时长、家庭看板、温和引导 |

**行动**：XIN-100（PM负责）— 多场景使用者画像设计

### 8.2 老年康复馆细分方向

Board洞察：刚退休的人群是老年康复馆方向的核心客户。
- "人，生来平凡；都有一颗不甘日常的心"
- 既对健康有需求，又可以释放被束缚的心
- 终于可以开始追求人生的感觉

**行动**：XIN-99（Researcher负责）— 老年康复馆市场深度研究

### 8.3 制造商切入逻辑修正

Board纠正：不要从"我们的产品多好"出发，要从"制造商在乎什么"出发。

核心认知：
- 制造商**根本不在乎**我们的数字底座
- 制造商在乎的是：**如何建立销售渠道，如何把货卖出去**
- 不同工厂在不同阶段，在乎的事情不一样
- 我们的价值主张必须从满足制造商卖货需求切入

**行动**：XIN-101（BD Manager负责）— 制造商切入策略重写

### 8.4 战略含义

Board补充对v1.1执行的影响：

| 原方向 | Board补充修正 | 影响程度 |
|--------|-------------|---------|
| 技术驱动的产品价值主张 | 需叠加制造商卖货视角的价值主张 | **重大** — BD话术和Demo需要重新包装 |
| 泛化的"健身器械数字化" | 需要分场景（健身房/康复馆/家庭）差异化设计 | **中等** — 产品架构已支持，需细化UI/UX |
| BD主打技术先进性 | BD应主打"帮你卖更多货" | **重大** — BD策略核心叙事要变 |
| 老年方向模糊 | 聚焦刚退休人群（55-70岁），情感激励+不甘日常 | **明确** — 缩窄靶心 |
