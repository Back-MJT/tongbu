## 执行摘要

行为改变理论是干预引擎的"软件逻辑"——再好的运动处方或睡眠建议，如果用户不执行，就毫无价值。本简报覆盖两个核心框架：

1. **COM-B模型**（Michie et al., 2011）：理解行为发生的机制，诊断行为障碍
2. **健康行动过程取向（HAPA）**（Schwarzer, 1992/2008）：设计分阶段干预策略

**关键发现**:
- COM-B的"能力×机会×动机"框架可直接映射到可穿戴设备数据和干预触发点
- HAPA的三阶段（动机阶段→意志阶段→行动阶段）完美匹配数字健康干预的典型用户旅程
- 两者结合：COM-B诊断行为障碍，HAPA设计转化干预，是目前数字健康领域证据最充分的组合

## 1. COM-B模型

### 1.1 框架来源

Michie et al. (2011) 在《British Journal of Health Psychology》提出COM-B模型，是英国国家健康服务体系（NHS）行为改变循证指南的理论基础。该框架被WHO和欧洲健康干预指南广泛采用。

### 1.2 三要素

```
行为 (Behavior) = 能力 (Capability) × 机会 (Opportunity) × 动机 (Motivation)
```

| 要素 | 子类型 | 定义 | 可穿戴设备映射 |
|------|--------|------|--------------|
| **能力 C** | 身体能力 | 体能、技能、健康状况允许执行行为 | 心肺耐力、体力活动水平、既往运动经验 |
| **能力 C** | 心理能力 | 知识、理解力、认知功能 | 用户健康素养、睡眠充足度（认知资源） |
| **机会 O** | 物理机会 | 环境允许行为发生的时间/空间/工具 | 设备可用性、健身房可达性、运动时间安排 |
| **机会 O** | 社会机会 | 社会规范、文化期待、他人支持 | 社交连接、社区归属、运动教练/伙伴 |
| **动机 M** | 反射性动机 | 即时欲望、习惯、情感反应 | 晨起精力、锻炼后愉悦感（多巴胺反馈） |
| **动机 M** | 反思性动机 | 计划、意图、价值取向 | 健康目标设定、干预依从性意图 |

### 1.3 干预功能（来自BCW框架）

COM-B配套的**行为改变Wheel（BCW）**定义了9种干预功能和9种政策类别。数字健康最相关的：

| 干预功能 | 策略 | 数字实现 | 证据等级 |
|---------|------|---------|---------|
| **教育** | 增加知识 | 推送健康信息、睡眠卫生教育 | 中（RCT） |
| **说服** | 使用情感激励 | 运动成就徽章、睡眠分数变化可视化 | 中（RCT） |
| **激励** | 奖励目标达成 | 积分、虚拟奖励、社交分享 | 强（RCT） |
| **训练** | 教授技能 | 动作示范视频、渐进式训练计划 | 强（RCT） |
| **限制** | 减少不当行为机会 | 睡前设备使用提醒（蓝光限制） | 中（队列研究） |
| **环境重构** | 改变物理环境 | 运动时间提醒、居家健身设备推荐 | 新兴 |
| **建模** | 示范行为 | 社交比较、运动圈/KOL内容 | 中（RCT） |
| **实现** | 直接帮助 | AI教练、个性化反馈 | 强（系统评价） |

### 1.4 行为障碍诊断检查清单

对每位用户，可基于COM-B诊断行为障碍：

```
运动干预场景诊断:
□ 能力不足: 心肺耐力不足？关节疼痛限制？缺乏运动技能？
□ 机会不足: 没有运动时间？附近无安全运动场所？缺少运动伙伴？
□ 动机不足: 对运动价值认知不足？有过失败经历？缺乏即时反馈？

睡眠干预场景诊断:
□ 能力不足: 不了解睡眠卫生知识？无法识别压力-睡眠关系？
□ 机会不足: 作息时间不规律？卧室环境不适合睡眠（光线/噪音）？
□ 动机不足: 晚间优先看手机而非睡眠？认为睡眠可以"补"回来？
```

### 1.5 算法参数建议

```python
# COM-B行为评分（0-100）
capability_score = (
    physical_capability * 0.5 +  # 心肺/肌力评估
    psychological_capability * 0.5  # 健康素养评估
) * equipment_accuracy_factor

opportunity_score = (
    physical_opportunity * 0.4 +  # 时间/空间/设备可用性
    social_opportunity * 0.6     # 社会支持权重更高
)

motivation_score = (
    reflective_motivation * 0.4 +  # 目标清晰度
    reflexive_motivation * 0.6     # 即时感觉权重更高
)

behavior_likelihood = (
    capability_score * 0.2 +
    opportunity_score * 0.3 +
    motivation_score * 0.5  # 动机权重最高
)

# 障碍诊断优先级排序
deficits = {
    'capability': 1 - capability_score,
    'opportunity': 1 - opportunity_score,
    'motivation': 1 - motivation_score
}
primary_deficit = max(deficits, key=deficits.get)
```

## 2. HAPA模型

### 2.1 框架来源

Schwarzer (1992) 提出，2008年完整版发表。是欧洲健康心理学最广泛使用的行为改变模型，也是WHO《健康促进全球词汇表》收录的推荐框架。

### 2.2 三阶段结构

```
┌─────────────────────────────────────────────────────────┐
│  动机阶段            │  意志阶段           │  行动阶段  │
│  (Motivation)        │  (Volition)         │  (Action)  │
├──────────────────────┼────────────────────┼────────────┤
│ 风险感知             │ 行动规划            │ 维持性    │
│  结果预期 (+/-)      │  应对规划            │  自我监控  │
│  自我效能 (+/-) ──┐  │  自我效能 (+/-)     │  社会支持  │
│                 │  │                     │  复因处理  │
│  行动意图 (+/-) ──┘  └────────────────────┘  成功体验  │
└─────────────────────────────────────────────────────────┘
```

### 2.3 核心构念

| 构念 | 定义 | 预测力 | 数字健康应用 |
|------|------|--------|------------|
| **风险感知** | 对健康风险的认知 | 中 | 睡眠债务可视化、运动对心脏的好处 |
| **结果预期** | 行为会带来正面结果的信念 | 强（Meta分析） | 预期收益展示（"跑步3月可降低血压"） |
| **自我效能** | 执行行为的能力信心 | 最强（r=0.4-0.6） | 渐进式任务设计、榜样示范 |
| **行动意图** | 决定开始行为的计划 | 意图-行为差距大 | 需要意志策略辅助 |
| **行动规划** | 何时/何地/如何行动的详细计划 | 强（缩小意图-行为差距） | 推送"今晚8点客厅跑步机" |
| **应对规划** | 遇到障碍时的应对策略 | 强（防止意图退化） | 障碍预案提示（"下雨替代方案"） |
| **自我监控** | 追踪自己行为 | 强（但需配合反馈） | 设备数据自动记录、每日打卡 |

### 2.4 阶段特异性干预

| 用户阶段 | 核心需求 | 干预重点 | 数字策略 |
|---------|---------|---------|---------|
| **前意图** | 风险感知 + 结果预期 | 激发动机，不急于规划 | 健康风险教育、他人成功故事 |
| **意图形成** | 自我效能 + 行动意图 | 建立信心，形成具体计划 | 小目标设计、成功体验积累 |
| **行动阶段** | 行动规划 + 应对规划 | 执行支持和障碍预防 | 提醒推送、障碍预案、社交激励 |
| **维持阶段** | 自我监控 + 复发应对 | 防止退步，建立习惯 | 长期趋势展示、障碍应对、回归快速重启 |

### 2.5 意图-行为差距（Intention-Behavior Gap）

HAPA的核心贡献之一是识别意图-行为差距。数据显示：
- 50-70%的人有运动意图，但实际坚持者仅20-30%
- **意图越高，行为执行率不一定越高**——需要意志策略

关键缓解手段：
1. **行动规划**（Implementation Intention）：将"我要多运动"转化为"我每周一/三/五18:00在客厅跑步机跑步30分钟"——格式固定效应（ Gollwitzer, 1999）
2. **应对规划**（Coping Planning）：预判障碍并制定应对策略

### 2.6 算法参数建议

```python
# HAPA阶段评估（基于用户行为数据）
def assess_hapa_stage(user_data):
    if user_data['completed_sessions'] == 0 and user_data['intention_score'] < 0.5:
        return 'preintention'
    elif user_data['completed_sessions'] == 0 and user_data['intention_score'] >= 0.5:
        return 'intention'
    elif user_data['completed_sessions'] < 8:  # 约4周习惯化期
        return 'action'
    else:
        return 'maintenance'

# 阶段特异性干预选择
HAPA_INTERVENTIONS = {
    'preintention': ['risk_literacy', 'outcome_benefits', 'social_proof'],
    'intention': ['self_efficacy_boost', 'action_planning', 'small_wins'],
    'action': ['implementation_intention', 'coping_planning', 'reminders'],
    'maintenance': ['self_monitoring', 'habit_formation', 'relapse_prevention']
}
```

## 3. COM-B × HAPA 整合应用

### 3.1 整合框架

```
第一步：用COM-B诊断"为什么不能做"
第二步：用HAPA决定"如何帮助用户开始并坚持"
```

| 场景 | COM-B诊断 | HAPA阶段 | 推荐干预 |
|------|---------|---------|---------|
| 首次用户，无运动习惯 | 动机不足（反射性+反思性均低）| 前意图 | 风险感知教育、成功故事 |
| 有运动意图但坚持不了 | 机会不足（无规律时间）| 意图形成 | 行动规划（时间地点固定） |
| 运动频率下降 | 动机不足（反射性：无聊）| 行动/维持 | 变化激励、新目标设定 |
| 睡眠干预配合度低 | 机会不足（睡前习惯难改）| 前意图 | 光照环境改造、限制策略 |

### 3.2 行为改变技术（BCTs）映射

基于Michie et al. (2013) 的BCTv1 taxonomy，以下是最有效的行为改变技术（BCTs），按证据强度排序：

| BCT | 描述 | 证据强度 | COM-B机制 | HAPA阶段 |
|-----|------|---------|-----------|---------|
| 1.2 | 利益自我交谈 | 强 | 动机（反射性）| 意图/行动 |
| 1.4 | 行动规划 | 强 | 动机（反思性）+ 机会 | 意图/行动 |
| 2.2 | 反馈示范 | 强 | 动机（反射性）| 行动/维持 |
| 2.3 | 自我监控 | 强 | 能力（心理）| 行动/维持 |
| 3.1 | 社交支持（实际）| 强 | 机会（社会）| 全阶段 |
| 4.1 | 循序渐进 | 强 | 能力+动机 | 行动/维持 |
| 8.1 | 技能教授 | 强 | 能力（身体+心理）| 行动 |
| 8.3 | 习惯形成 | 强 | 动机（反射性）| 维持 |
| 9.1 | credible source | 强 | 动机（反思性）| 前意图/意图 |
| 10.4 | 行为奖励 | 强 | 动机（反射性）| 行动/维持 |
| 15.1 | 社交比较 | 中 | 动机（反思性）| 意图/行动 |

```
用户档案增加字段:

干预触发规则:

- 睡眠效率提升 → 触发结果预期强化（"你睡得好，今天状态好"）
```

## 4. 证据评价与局限性

### 4.1 证据质量

| 框架 | 证据基础 | 主要局限 |
|------|---------|---------|
| COM-B | 中-强（多项系统评价）| 来自西方健康人群，跨文化适用性待验证 |
| HAPA | 强（50+国家，200+研究）| 中国人群数据相对较少 |

### 4.2 文化适用性注意事项

- HAPA的"自我效能"概念在集体主义文化中可能表现形式不同（社会自我效能可能比个体自我效能更有效）
- 中国老年人群中，社会支持（家庭督促）对运动依从性的影响可能比西方国家更强
- 运动干预中的"面子"因素——公开运动承诺可能产生更大的社会压力和依从性
- **建议**：产品设计应增加"家庭/社群"维度的社会支持模块

### 4.3 局限性

- COM-B/HAPA均主要适用于有意愿改变的用户，对于完全无动机用户（"stage of change"前意识阶段）干预效果有限
- 行为改变理论在慢病管理（糖尿病、高血压）中的证据比健康人群更强
- 数字化BCI（行为改变干预）实施的质量（fidelity）是重要调节变量

## 5. BD科学话术

### 话术1: "为什么用户不坚持"

> 即使用户认同运动有益，也往往无法坚持。行为科学研究表明，50-70%的人有运动意图，但实际坚持者仅20-30%。根本原因通常不是意志力问题，而是三个可诊断的障碍：能力不足（不知道怎么做）、机会不足（没有时间/工具）、动机不足（没有即时反馈）。昕动智能的干预引擎内置COM-B行为诊断，自动识别每位用户的核心障碍，并匹配合适的行为改变策略。

### 话术2: "从'想运动'到'在运动'"

> 我们不只是给出运动处方——我们用HAPA行为科学框架，将用户从"有意图"推进到"在行动"。关键是把"我要多运动"这样的模糊意图，转化为"每周一/三/五晚上8点在家跑步机跑30分钟"的具体计划。数字健康研究证明，这种行动规划（Implementation Intention）可将执行率提高2-3倍。

### 话术3: "睡眠也是行为"

> 很多人以为睡眠问题是生理问题，实际上睡前行为（如手机使用、咖啡因摄入、光线暴露）是可控的行为因素。昕动智能的睡眠干预结合刺激控制疗法和睡眠限制疗法，通过改变睡前行为习惯来改善睡眠质量——这是国际睡眠医学界推荐的一线方案（CBT-I）。
