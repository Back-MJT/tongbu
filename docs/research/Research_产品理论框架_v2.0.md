HealthHub产品的本质是**将人类数十年的运动科学、健康医学研究转化为可计算的个性化处方引擎**。这套理论框架是我们四层产品的科学基础：

| 产品层 | 理论框架作用 |
|--------|------------|
| Layer 0: IMU+BLE硬件 | 从传感器原始数据 → 关节力学/运动质量 |

## 一、运动生理学框架

### 1.1 心率与心肺功能

#### 框架1: 最大心率预测 — Tanaka公式

```
输入: 年龄（age, 岁）
输出: 预测最大心率（HRmax, bpm）
公式: HRmax = 208 - 0.7 × age
适用: 健康成人（20-70岁），误差±10 bpm
证据等级: Strong（多人群跨种族验证，r²=0.60-0.70）
来源: Tanaka et al. (2001), J Appl Physiol
```

**产品映射**: Device SDK层，从用户年龄自动计算靶心率区间（Zone 2/3/4/5）

**局限性**: 
- 不适用于心脏病患者（需实测)
- 高血压患者可能偏低5-8 bpm

**变体**:
- 青少年: HRmax = 220 - age（传统公式，证据较弱）
- 挪威研究(2010): HRmax = 211 - 0.64×age（女性表现更好）

---

#### 框架2: 储备心率法（Karvonen Formula）— 个性化靶心率

```
输入: 年龄、最大心率（实测或估算）、静息心率（HRrest）
输出: 靶心率（THR）= (HRmax - HRrest) × 强度% + HRrest
证据等级: Strong（ACSM 2021立场声明推荐）
适用: 运动处方金标准方法
```

**HealthHub产品计算**:
```
Zone 2 (脂肪燃烧/耐力): THR = (HRmax - HRrest) × 0.60-0.70 + HRrest
Zone 3 (有氧): THR = (HRmax - HRrest) × 0.70-0.80 + HRrest  
Zone 4 (无氧阈): THR = (HRmax - HRrest) × 0.80-0.90 + HRrest
Zone 5 (最大): THR = (HRmax - HRrest) × 0.90-1.00 + HRrest
```

---

#### 框架3: 最大摄氧量（VO2max）估算

```
方法A — 2km Walk Test（老年人/初级）:
输入: 步行时间(秒)、心率(次)、年龄、性别、体重(kg)
公式: VO2max = 100.07 + 0.16395×体重 - 0.1927×年龄 + 0.7847×性别编码 - 1.2976×心率
来源: Ebbeling et al. (1991), Med Sci Sports Exerc

方法B — 预测公式（基于心率和年龄）:
VO2max = 15.3 × (HRmax / HRrest)
来源: Winchester et al. (2009), J.Strength Cond Res
误差: ±15-20%

方法C — Rockport Fitness Walking Test:
VO2max = 132.853 - (0.0769 × 体重lbs) - (0.3877 × 年龄) + (6.315 × 性别) - (3.2649 × 时间) - (0.1565 × 心率)
来源: Kline et al. (1987)
```

**产品意义**: Layer 2训练引擎用VO2max估算值作为运动强度分级的客观依据

---

#### 框架4: 主观疲劳感（RPE）— Borg量表

```
输入: 用户主观疲劳评分（6-20分量表）
输出: 推断心率百分比；运动强度自评
公式逆推: 推断心率 = RPE × 10 - 10（适用于RPE 12-16）
证据等级: Strong（Meta分析，r=0.62-0.80与心率相关）

Borg RPE → 靶心率映射（ACSM）:
RPE 6-8: 非常轻松 → Zone 1 (<50% HRmax)
RPE 9-10: 轻松 → Zone 1-2 (50-60%)
RPE 11-12: 略难 → Zone 2 (60-70%)
RPE 13-14: 略难+ → Zone 3 (70-80%)
RPE 15-16: 难 → Zone 4 (80-90%)
RPE 17-18: 非常难 → Zone 5 (90-100%)
RPE 19-20: 极度用力 → 超过最大
```

**产品映射**: 小程序用户自评输入 → 交叉验证心率数据 → 调整处方强度

### 1.2 能量消耗与代谢

#### 框架5: 静息代谢率（RMR）估算

```
公式: RMR = 370 + (21.6 × 瘦体重kg)
来源: Cunningham (1980), Am J Clin Nutr

变体（Mifflin-St Jeor，更精确）:
男性: RMR = (10 × 体重kg) + (6.25 × 身高cm) - (5 × 年龄) + 5
女性: RMR = (10 × 体重kg) + (6.25 × 身高cm) - (5 × 年龄) - 161
证据等级: Strong（与间接测热法相关性r=0.86-0.90）
误差: ±10-15%
```

**产品映射**: 运动后EPOC（过量氧耗）估算、每日总能量消耗（DEE）计算

---

#### 框架6: 运动能量消耗（科罗门空气置换原理简化）

```
输入: 体重(kg)、运动时间(分钟)、MET值（代谢当量）
公式: 能量消耗(kcal) = 体重(kg) × MET × 时间(h)

主要运动MET值:
- 静坐: 1.0 MET = RMR
- 步行(4.5km/h): 3.0 MET
- 骑行(16km/h): 7.0 MET
- 跑步(8km/h): 9.0 MET
- 跑步(10km/h): 11.0 MET
- 力量训练(中等强度): 4.0-6.0 MET
来源: Ainsworth et al. (2011), Med Sci Sports Exerc

准确性: 误差±20-30%（大规模应用可接受）
证据等级: Moderate（群体水平准确，个体差异大）
```

**产品意义**: Layer 1 Device SDK计算每次运动的kcal消耗（已知运动类型+时间+体重）

### 1.3 运动处方核心框架

#### 框架7: FITT-VP原则（运动处方黄金框架）

```
FITT-VP = Frequency(频率) × Intensity(强度) × Time(时间) × Type(类型) × Volume(容量) × Progression(进阶)

Frequency（频率）:
  - 有氧: 3-5天/周（Zone 2/3），2-3天（高强度）
  - 力量: 2-3天/周（非连续）
  - 柔韧: 2-3天/周

Intensity（强度）:
  - 有氧: %HRmax / %VO2max / RPE
  - 力量: %1RM / RPE

Time（时间）:
  - 有氧: 150-300分钟/周（中等强度）或75-150分钟（高强度）
  - 力量: 每肌群8-12次×2-4组

Type（类型）:
  - 有氧: 步行、骑行、游泳等
  - 力量: 抗阻、自由重量、器械

来源: ACSM Guidelines (11th Edition, 2021)
证据等级: Strong
```

**产品映射**: Layer 2训练引擎处方生成的核心逻辑框架

---

#### 框架8: 2区间/3区间训练模型（Polarized Training）

```
训练强度分布（Seiler分区）:

Zone 1 (低强度/LIT): HRmax的60-72%，RPE 1-2/10
  → 训练量占比: 80%（恢复/基础）
  
Zone 2 (中等/MIT): HRmax的72-87%，RPE 3-4/10
  → 训练量占比: ~0%（竞技运动员少量，健康人群可适当提高）

Zone 3 (高强度/HIT): HRmax的87-95%，RPE 7-8/10
  → 训练量占比: 20%

 polarized训练（竞技运动员）:
  - 高强度只做Zone 3，无Zone 2
  - 健康人群: Zone 1 + Zone 3

来源: Stöggl & Sperlich (2014), Front Physiol
证据等级: Strong（竞技运动），Moderate（健康/康复）
```

**产品映射**: Layer 2训练引擎用Polarized模型做运动处方类型选择

## 二、生物力学与IMU运动捕捉框架

### 2.1 关节角度与运动质量

#### 框架9: IMU关节角度估算（已验证）

```
输入: IMU三轴加速度 + 三轴角速度
输出: 关节角度（°）、运动阶段（向心/离心）

核心算法:
1. 陀螺仪积分: θ(t) = θ(0) + ∫ω(t)dt（漂移累积，需滤波）
2. 互补滤波: θ = α×(θ_gyro) + (1-α)×(θ_acc)
   α = 0.98（高频用陀螺仪，低频用加速度计）
3. 零角度标定: 静止时自动校准

精度指标（Sato et al. 2023）:
- RMSE = 3.2°（矢状面）→ 临床可接受<5°
- 动作阶段识别灵敏度: 91-94%
- 与光学系统ICC = 0.94

证据等级: Strong（多个独立研究验证）
```

**产品映射**: Layer 0（IMU固件）+ Layer 1（Device SDK）→ 运动质量评分

---

#### 框架10: 重复次数计数（Rep Counting）

```
输入: IMU角速度波形（峰值检测）
输出: 重复次数

算法:
- 设定角速度阈值（阈值法）
- 或: ML模型（CNN/LSTM）识别动作模式
- 检测每次重复的起始点和结束点

精度:
- 自由重量训练: 准确率96.2%（Harm et al. 2022）
- 划船机: 准确率94.7%
- 跑步机步数: 误差<2%

证据等级: Strong
```

**产品映射**: 力量训练处方执行监控（是否完成规定次数）

---

#### 框架11: 左右侧不对称检测

```
输入: 左右侧IMU数据同步采集
输出: 不对称指数（%）

公式: 不对称% = |左侧值 - 右侧值| / 最大值 × 100

典型阈值:
- <5%: 正常
- 5-10%: 轻度不对称（注意监控）
- >10%: 显著不对称（调整处方）
- >15%: 可能损伤风险

证据等级: Moderate（特定人群有效，普通人群参考意义有限）

来源: Rogowski et al. (2022), J Sports Sci
```

---

#### 框架12: 运动速度与功率估算

```
输入: IMU跟踪的杆/器械位移速度
输出: 平均速度、峰值速度、功率(W)

公式:
- 平均速度 = 位移 / 时间
- 峰值速度 = max(速度)
- 功率 ≈ 力 × 速度（简化版）

准确性:
- 速度误差 <5%（与测速计对比）
- 功率估算误差 10-15%（不如测力平台，但成本低得多）

应用: 力量训练负荷监控（避免速度衰减过多导致过度训练）
证据等级: Moderate-High
```

## 三、行为改变理论框架

### 3.1 阶段式行为改变模型

#### 框架13: 健康行动过程取向（HAPA）

```
HAPA三阶段模型（Schwarzer 1992, 2008）:

阶段1: 前意向阶段（Preintention）
  → 用户尚未考虑运动
  → 干预: 风险感知教育 + 结果预期引导

阶段2: 意向阶段（Intention）
  → 用户有意向但尚未行动
  → 干预: 行动规划 + 障碍识别 + 自我效能建立

阶段3: 行动阶段（Action）
  → 用户已在运动
  → 干预: 维持策略 + 自我监控 + 社会支持

关键构念:
- 结果预期: 运动能带来什么好处（结果预期高→意向强）
- 感知障碍: 认为有什么困难（感知障碍低→意向强）
- 自我效能: 有信心克服障碍（核心预测变量）
- 行动规划: 具体何时何地如何运动（If-then plan）

证据等级: Strong（Meta分析，n=30000+，意向-行为相关性r=0.37）
```

**产品映射**: Layer 2训练引擎，根据用户所处HAPA阶段推送不同类型干预内容

---

#### 框架14: COM-B行为系统

```
COM-B = Capability(能力) × Motivation(动机) × Opportunity(机会)

C - 身体能力（Physical Capability）: 体能、健康状况
M - 心理动机（Reflective + Automatic Motivation）: 习惯、情绪、价值观
O - 物理机会（Physical Opportunity）: 时间、设备、可及性
O - 社会机会（Social Opportunity）: 社会支持、文化规范

行为发生条件: C×M×O同时激活

干预功能（BCW框架, Michie 2011）:
- 教育（Education）→ 增加知识
- 劝导（Persuasion）→ 建立动机
- 训练（Training）→ 提升技能
- 限制（Restriction）→ 减少机会
- 环境重构（Environmental restructuring）→ 改变物理/社会环境
- 建模（Modeling）→ 榜样示范
- 使能（Enablement）→ 降低障碍

证据等级: Strong（WHO行为改变方法论基础）
```

---

#### 框架15: 习惯形成理论（Habit Formation）

```
习惯形成条件:
1. 重复（Repetition）: 同一行为在相同情境下重复
2. 奖励（Reward）: 行为后获得即时正向反馈
3. 情境线索（Cue）: 在固定时间/地点触发行为

典型习惯周期: 21-66天（中位数40天，实际分布很广）
证据等级: Moderate（习惯测量方法学争议较大）

应用:
- 固定的运动时间（情境线索）
- 运动后即时数据反馈（即时奖励）
- 渐进式从小目标开始（降低阻力）

来源: Lally et al. (2010), Br J Health Psychol
```

**产品映射**: 小程序推送时机设计、即时反馈激励机制

## 四、睡眠科学框架

### 4.1 睡眠结构与评估

#### 框架16: 睡眠阶段分类（NASPE 2007）

```
睡眠结构:

N1（入睡期）: 占总睡眠5%
  - 脑电: θ波(4-8Hz)，低电压混合频率
  - 容易被唤醒
  - 标志: 出现睡眠纺锤波/K复合波

N2（浅睡期）: 占总睡眠45-55%
  - 脑电: 睡眠纺锤波(12-14Hz)+K复合波
  - 体温下降、心率减慢
  - 记忆巩固重要阶段

N3（深睡期/慢波睡眠）: 占总睡眠15-20%（年轻人）
  - 脑电: δ波(0.5-2Hz)，高电压慢波
  - 生长激素分泌高峰（70%在N3期）
  - 免疫修复、身体恢复
  - 随着年龄增长显著减少（60岁+几乎消失）

REM（快速眼动期）: 占总睡眠20-25%
  - 脑电: 近似清醒的β波，混θ波
  - 做梦、情绪记忆整合
  - 体温调节丧失、心率变异性高

来源: Iber et al. (2007), AASM睡眠分期指南
证据等级: Strong（金标准：多导睡眠图PSG）
```

**产品映射**: 消费级可穿戴设备可识别N1/N2/N3/REM，精度60-70%（证据简报已覆盖）

---

#### 框架17: 睡眠质量评估模型

```
输入: 总睡眠时间、入睡时间(TST)、睡眠效率、觉醒次数
输出: 睡眠质量评分

睡眠效率 = 总睡眠时间 / 卧床时间 × 100%
  - >85%: 良好
  - 75-85%: 一般
  - <75%: 睡眠障碍指征

PSQI（Pittsburgh Sleep Quality Index）:
- 7项自评维度，总分0-21
- >5分: 睡眠质量差
- 临床上用于筛查
- 证据等级: Strong（中文版已验证，Cronbach's α=0.82）

ISI（失眠严重程度指数）:
- 7项，总分0-28
- 0-7: 无失眠
- 8-14: 亚阈值失眠
- 15-21: 中度失眠
- 22-28: 重度失眠

来源: Buysse et al. (1989) PSQI; Bastien et al. (2001) ISI
证据等级: Strong
```

### 4.2 睡眠优化干预

#### 框架18: CBT-I（失眠认知行为疗法）核心组件

```
CBT-I五组件:

1. 睡眠限制疗法（Sleep Restriction）
   → 限制卧床时间至实际睡眠时间（最低4.5h）
   → 睡眠效率>85%后逐步增加卧床时间
   
2. 刺激控制（Stimulus Control）
   → 仅在困倦时上床
   → 20分钟未入睡则离开床
   → 固定起床时间（即使睡眠不足）
   → 避免白天小睡

3. 认知重构（Cognitive Restructuring）
   → 识别并挑战关于睡眠的负面信念
   → "我必须睡够8小时"等灾难化思维

4. 睡眠卫生教育（Sleep Hygiene）
   → 睡前限制蓝光/咖啡因
   → 卧室温度18-20°C最佳
   → 规律运动（但非睡前3小时内）

5. 放松训练（Relaxation）
   → 渐进式肌肉放松
   → 呼吸训练
   → 正念冥想

疗效: 与药物等效，长期效果优于药物
证据等级: Strong（RCT+Meta分析，ACP 2016推荐）

来源: Morin et al. (2006), JAMA; Qaseem et al. 2016
```

**产品映射**: Layer 2训练引擎的睡眠处方模块，基于CBT-I原理

## 五、心血管健康与HRV框架

### 5.1 心率变异性（HRV）科学

#### 框架19: HRV时域指标

```
输入: 连续RR间期（心电或光电容积图）
输出: SDNN, RMSSD, pNN50等

主要指标:
- SDNN（正常RR间期标准差）: 全因死亡风险预测指标
  * 24h SDNN <50ms: 高风险
  * 24h SDNN >100ms: 低风险
  * 5min静息SDNN与24h SDNN高度相关(r=0.85)

- RMSSD（相邻RR间期差值均方根）: 副交感神经活性金指标
  * 短时分析（5min）: RMSSD >30ms（年轻人）表示良好恢复
  * 运动后RMSSD恢复速度：恢复越快→体能越好

- pNN50: 相邻RR间期差>50ms的比例
  * 与RMSSD高度相关(r>0.90)

证据等级: Strong（流行病学验证）
```

---

#### 框架20: HRV频域指标

```
输入: RR间期序列 → FFT/自回归分析
输出: 功率谱密度

频段:
- VLF（0.003-0.04Hz）: 生理意义不完全明确，与自主神经系统整体状态相关
- LF（0.04-0.15Hz）: 交感+副交感（以交感为主）
  * 增加: 压力、焦虑、代谢需求
  * 降低: 静息、健康训练状态

- HF（0.15-0.40Hz）: 副交感/迷走神经（呼吸调制）
  * 增加: 放松、恢复、迷走激活
  * 降低: 压力、焦虑、运动

- LF/HF比值: 交感-迷走平衡
  * 注意: 受姿势、呼吸深度影响大，个体差异显著
  * 单一指标解读需谨慎

证据等级: Strong（实验生理学），Moderate（临床应用）
```

---

#### 框架21: HRV压力检测算法

```
输入: 短时RR数据（30秒-5分钟）
输出: 压力状态分类（正常/轻度/中度/高度）

Kim et al. (2018)算法:
- 使用RMSSD和α1（心率复杂性指标）
- 准确率: 78-85%（vs 主观问卷）
- 适合实时检测

基于HRV的恢复评分（Firstbeat等商业算法）:
恢复分 = f(夜间RMSSD, HRV恢复速度, 静息心率)
范围: 0-100

证据等级: Moderate-High
```

**产品映射**: Layer 1 Device SDK计算HRV指标，Layer 2引擎用HRV做压力处方触发

## 六、压力与心理健康框架

#### 框架22: 感知压力量表（PSS-10）

```
输入: 10项自评问题（过去一个月的主观压力感受）
输出: PSS总分0-40

评分:
- 0-13: 低压力
- 14-26: 中等压力
- 27-40: 高压力

心理测量:
- Cronbach's α = 0.78-0.91
- 2周重测信度: r = 0.85
- 与皮质醇水平弱相关（r=0.15-0.30），说明感知压力≠生理压力

来源: Cohen et al. (1988), Health Psychology
证据等级: Strong（最广泛使用的压力自评工具）
```

#### 框架23: 正念减压（MBSR）与呼吸训练效果

```
正念减压循证效果:
- 疼痛感知降低: Cohen's d = 0.52（Meta分析）
- 焦虑改善: Cohen's d = 0.40
- 抑郁复发减少: HR = 0.69（与药物相当）
- 皮质醇水平: 降低约15%（短期项目）

呼吸训练降压机制:
- 缓慢呼气（6秒）→ 激活迷走神经 → 副交感激活 → 心率↓血压↓
- 3-5分钟Box Breathing: 可降低收缩压5-10mmHg
- 证据等级: Strong（随机对照试验支持）

来源: Grossman et al. (2004), Br J Health Psychol; Ma et al. (2017), Sci Rep
```

## 七、整合框架：多维度健康评分

#### 框架24: 综合健康评分算法（HealthHub自建）

基于以上框架，HealthHub Layer 3数据价值层整合多维数据：

```
健康评分 = w1×运动分 + w2×睡眠分 + w3×心肺分 + w4×恢复分

子维度:
- 运动分 = f(完成度, 强度达标, 训练多样性)
- 睡眠分 = f(睡眠效率, 睡眠时长, 规律性)
- 心肺分 = f(静息心率趋势, HRV恢复, 活动水平)
- 恢复分 = f(HRV指标, 主观疲劳感, 睡眠质量)

权重说明:
- w1-w4根据用户健康目标动态调整
- 减重目标→运动分权重高
- 压力管理目标→恢复分权重高
- 心脏康复→心肺分权重高

数据基础: 所有子维度均来自上述已验证框架的组合输出
```

## 八、证据等级评级标准

本文档采用以下证据等级体系：

| 等级 | 定义 | 典型来源 |
|------|------|---------|
| **Strong** | 多项独立RCT/Meta分析一致，且效应量明确 | ACSM立场声明、WHO指南、大型Meta分析 |
| **Moderate** | 多项研究支持，但存在争议或局限性 | 单一大型RCT、多个小型RCT |
| **Moderate-Low** | 初步研究支持，需要更多验证 | 小规模RCT、队列研究 |
| **Emerging** | 初步研究，机制明确但效果待验证 | Pilot RCT、动物实验、专家共识 |
| **Theoretical** | 理论推导，尚无充分实证 | 机制研究、理论模型 |

## 九、关键证据缺口与未来研究方向

### 9.1 当前产品的证据盲区

| 盲区 | 说明 | 优先级 |
|------|------|--------|
| 中国人群特异性公式 | 大多数公式基于欧美人群，中国人最大心率可能低5-8 bpm | 高 |
| 老年人IMU精度 | 现有IMU精度数据来自健康成人，老年人关节活动度差异未充分研究 | 中 |
| 复合健康评分算法验证 | 自建综合评分缺乏临床验证 | 高 |
| HRV恢复与运动处方的动态关系 | 当前为规则估算，缺乏前瞻性验证 | 中 |
| 营养框架缺失 | 仅覆盖运动/睡眠/压力，营养处方尚未建立理论框架 | 低（Phase 2） |
