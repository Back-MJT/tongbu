# 若依管理端重构清单

## 目标

将当前管理端统一收口到“器械 + IMU + 训练记录”这条真实业务主线：

- 小程序扫码的是**器械**
- 小程序连接的是**GY-BLE25T / IMU 设备**
- 后台展示的是**器械使用情况、IMU状态、训练记录**

### 当前前端问题

这些页面更偏“设备接入平台”视角，而不是“器械训练平台”视角。

当前还缺少一个正式的一层概念页：

- **器械管理**

也缺少训练记录主线页：

- **训练会话列表**
- **训练组明细 / 训练详情**
- **用户设备绑定关系**

## 当前后端现状

### 已落地

- 小程序扫码解析接口：
  - `GET /api/mini/equipment/resolve`
- 小程序训练会话写入：
  - `interv_session`
- 小程序训练组明细写入：
  - `xd_training_set`
- 器械与 IMU 关系表：
  - `xd_equipment`
  - `xd_equipment_device`

### 当前后端缺口

管理端还缺正式 CRUD / 列表接口：

1. 器械管理接口
2. 器械-IMU 绑定管理接口
3. 用户设备绑定列表接口
4. 训练会话管理接口
5. 训练组明细查询接口

这意味着：

**管理端下一步优先级不是先做更多 Vue 页面，而是先补后端接口。**

## 推荐的管理端结构

### 第一层菜单

建议保留并调整为：

1. `IoT看板`
2. `IMU设备管理`
3. `器械管理`
4. `设备分组`
5. `IMU数据`
6. `设备日志`
7. `训练记录`

### 页面职责建议

#### 1. IoT看板

保留现有页面，但统计口径改成真实训练主线：

- 今日训练人次
- 今日动作次数
- 活跃器械 TOP10
- 在线 IMU 数
- 器械使用率

#### 2. IMU设备管理

复用现有 `/iot/device`，但字段要更贴近 GY-BLE25T：

- `deviceCode`
- `deviceName`
- `bluetoothName`
- `serviceUuid`
- `notifyCharUuid`
- `firmwareVersion`
- `status`
- `lastSeenAt`

#### 3. 器械管理

新增页面，核心字段：

- `equipmentCode`
- `equipmentName`
- `equipmentType`
- `location`
- `qrContent`
- `status`
- 当前绑定 IMU

#### 4. 用户设备绑定

建议作为器械/训练辅助页，字段：

- `userId`
- `deviceCode`
- `deviceName`
- `boundAt`
- `status`

#### 5. 训练记录

新增训练记录主页面，字段：

- `sessionId`
- `userId`
- `equipmentCode`
- `deviceCode`
- `exerciseType`
- `completedSets`
- `totalReps`
- `durationMinutes`
- `sessionTime`

详情页展示：

- `sets[]`
- 每组 `reps / durationSec / startedAt / endedAt`

## 实施顺序

### Phase 1：先补后端接口

优先补若依后端接口：

1. `/iot/equipment/list`
2. `/iot/equipment/{id}`
3. `/iot/equipment`
4. `/iot/equipment/binding/*`
5. `/iot/training/session/list`
6. `/iot/training/session/{id}`
7. `/iot/user-device-binding/list`

### Phase 2：复用现有 IoT 页面

先不大改布局，只做最小对齐：

1. `/iot/device` 增加 BLE 字段展示
2. `/iot/dashboard` 调整统计文案
3. `/iot/imu` 保持为 IMU 原始/最近数据查看页

### Phase 3：新增业务主线页面

新增：

1. `器械管理`
2. `训练记录`
3. `训练详情`

## 建议的下一步

下一轮优先做：

1. 后端新增 `器械管理` Controller + Mapper + CRUD
2. 后端新增 `训练记录列表/详情` 查询接口
3. 管理端接入 `器械管理` 页面

这样我们能先把“扫码对象是器械”这件事在后台正式立住。
