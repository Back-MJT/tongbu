## 一、产品定位

面向器械厂商合作伙伴的 **AI 个性化运动处方系统**，通过微信小程序为终端用户
提供每日训练计划。核心卖点：

1. **器械感知** — 根据用户当前使用的器械类型（智能力量站/跑步机/椭圆机/划船机等）
   自动生成匹配的运动方案
2. **AI 私教** — 每组动作配有语音级私教指导（动作要领、呼吸节奏、安全提醒）
3. **心率引导** — 接入可穿戴设备数据后，自动计算个人化靶心率区间（基于 Karvonen 公式）
4. **循证依据** — 所有推荐基于 ACSM 2021 第7章 FITT 框架 + WHO 2020 体力活动指南

## 四、算法逻辑

### 4.1 用户分阶

| 阶段 | 条件 | 特征 |
|------|------|------|
| 初学期 | 训练<4次/月 | 减量20%，延长休息，选择较轻重量 |
| 成长期 | 4-12次/月 | 标准FITT参数 |
| 进阶期 | >12次/月 | 增加组数，缩短休息，渐进超负荷 |

### 4.2 FITT 参数调整

| 参数 | 初学期 | 成长期 | 进阶期 |
|------|--------|--------|--------|
| 组数 | 标准-1 | 标准 | 标准+1 |
| 负荷 | ×0.7 | 标准 | ×1.15 |
| 休息 | +15秒 | 标准 | -15秒 |
| 强度 | 低强度 | 中等强度 | 高强度 |

### 4.3 心率区间（Karvonen 公式）

```
靶心率 = (HRmax - 静息HR) × 强度百分比 + 静息HR
HRmax = 208 - 0.7 × 年龄  (Tanaka 2001)
```

| 强度 | %HRR | 适用人群 |
|------|------|----------|
| 低 | 40-55% | 初学期/高血压用户 |
| 中 | 55-70% | 成长期 |
| 高 | 70-85% | 进阶期 |

### 4.4 久坐人群保护

- 月训练<4次的用户：动作数自动裁剪至3个
- 新用户：自动推送「从微习惯开始」健康贴士


### 6.2 任务状态流转

```
pending → in_progress → completed
                    └→ skipped
```

前端在用户开始一组时设为 `in_progress`，完成全部组数后设为 `completed`。
