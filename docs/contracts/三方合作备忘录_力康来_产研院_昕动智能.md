## 第一条 合作背景与目的

### 1.1 合作背景

1. 甲方为宁津县健身器材制造企业，有意对其产品进行智能化升级，加装IMU（惯性测量单元）传感模组并接入数据采集与分析系统，以提升产品附加值及出口竞争力。

2. 丙方具备IMU传感模组、BLE（低功耗蓝牙）通信、数据平台及AI训练引擎的完整技术方案，可提供从硬件到软件的一站式数字化服务。

3. 乙方为当地唯一一家科学研究和技术服务业性质的民办非企业单位，由当地政府与河北工业大学合作建立，核心任务是服务宁津本地产业发展、推动县域经济数字化转型。乙方作为丙方与甲方之间的协调桥梁，为双方合作提供组织层面的信任支持。

### 1.2 合作目的

三方通过本备忘录明确合作意向、各方角色定位及阶段性安排，把事情说清楚，把风险分清楚，为后续正式合同的签署奠定基础。

## 第二条 合作模式与角色定位

### 2.1 合作架构

```
阶段一（开发期）:

  甲方（力康来）── 提出需求、提供设备 ──→ 乙方（产研院）
                                               │
                                    委托开发 + 开发费用
                                               │
                                               ↓
                                        丙方（昕动智能）
                                               │
                                    系统开发 + IMU安装部署
                                               │
                                               ↓
                                    乙方组织三方验收
                                               │
                                               ↓
阶段二（正式采购期）:

  甲方 ────── 直接签约 / SaaS订阅付费 ──────→ 丙方

  乙方退出商业关系，保留成果署名权及宣传权
```

### 2.2 角色定位

| 角色 | 职责 |
|------|------|
| **甲方**（力康来） | 提出业务需求，提供试点设备及测试环境，指定技术对接人，配合现场安装和数据验证，参与验收 |
| **乙方**（产研院） | 作为丙方与甲方之间的协调桥梁，提供组织信任支持；丙方与甲方之间的合作条款（包括费用安排）由双方自行协商，乙方不参与具体商业条款的拟定或执行；组织验收评审；不参与技术开发 |
| **丙方**（昕动智能） | 技术方案设计、IMU模组供应、软件开发、系统部署、技术培训及售后支持 |

### 2.3 核心原则

1. **系统归属**: 数字化系统的知识产权归丙方所有，甲方获得系统使用权。
2. **乙方角色**: 乙方作为协调桥梁，不参与具体技术开发，不分享知识产权，不承担商业担保责任。丙方与甲方之间的商业安排由双方自行协商。
3. **分段推进**: 开发阶段由乙方委托丙方并支付开发费用；开发完成后由甲方直接向丙方采购，乙方退出商业关系。
4. **本地扶持**: 乙方基于扶持本地企业的宗旨参与本合作。

## 第五条 数据归属与安全

### 5.1 数据归属

| 数据类型 | 归属方 | 使用限制 |
|---------|--------|---------|
| 设备运行原始数据 | 甲方 | 丙方经授权可使用，不得向第三方提供 |
| 脱敏聚合数据 | 丙方 | 可用于产品优化和行业报告，不暴露甲方及终端用户身份 |
| 终端用户训练行为数据 | 甲方（需用户授权） | 仅聚合统计使用，遵守PIPL |
| 系统算法和平台数据 | 丙方 | 含AI训练模型、行业基准数据 |

### 5.2 数据安全

1. 丙方确保数据存储和传输符合《中华人民共和国个人信息保护法》（PIPL）及《数据安全法》要求。
2. 甲方负责获取终端用户的数据采集授权。
3. 数据存储于中华人民共和国境内服务器。
