# 微信小程序真机调试指南

## 🎯 问题：小程序无法访问后端 API

### 原因
微信小程序在真机上运行时，会校验服务器域名：
- 必须是 HTTPS（不支持 HTTP）
- 必须在微信公众平台配置白名单
- 域名必须备案

### 解决方案

---

## 方案 1：开发阶段 - 使用"不校验合法域名"（推荐）

### 在微信开发者工具中

1. **打开项目设置**
   - 点击右上角"详情"
   - 找到"本地设置"
   - 勾选 ✅ **"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"**

2. **真机调试**
   - 点击工具栏的"真机调试"
   - 在弹出的选项中勾选 ✅ **"不校验合法域名"**
   - 用手机扫码打开小程序

3. **测试 API 连接**
   - 小程序应该可以访问 `http://192.168.1.102:8080`
   - 查看控制台日志确认 API 调用成功

### 注意事项
- ⚠️ 这个设置只在开发阶段有效
- ⚠️ 正式发布时必须配置 HTTPS 域名
- ⚠️ 手机和电脑必须在同一个 Wi-Fi 网络

---

## 方案 2：生产环境 - 配置服务器域名

### 步骤 1：准备 HTTPS 域名

**需要：**
- 一个已备案的域名（如：`api.xindong.health`）
- SSL 证书（可以使用 Let's Encrypt 免费证书）
- 服务器（用于部署后端）

### 步骤 2：在微信公众平台配置

1. **登录微信公众平台**
   - 访问：https://mp.weixin.qq.com/
   - 使用小程序账号登录

2. **配置服务器域名**
   - 进入：开发 → 开发管理 → 开发设置 → 服务器域名
   - 点击"修改"
   - 添加 **request 合法域名**：
     ```
     https://api.xindong.health
     ```
   - 点击"保存并提交"

3. **配置说明**
   - 域名必须是 HTTPS
   - 域名必须备案
   - 每月只能修改 5 次
   - 需要等待审核（通常几分钟）

### 步骤 3：部署后端到服务器

使用之前创建的 Docker 部署方案：

```bash
# 1. 上传项目到服务器
scp -r Xindong_Platform-main user@your-server:/path/to/

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 3. 一键部署
./deploy.sh
# 选择 "2) 生产环境"

# 4. 配置 Nginx 反向代理和 HTTPS
```

### 步骤 4：更新小程序配置

修改 `mini-program/src/config/env.ts`：

```typescript
production: {
  apiBase: 'https://api.xindong.health',
  ieBase: 'https://api.xindong.health',
  wsUrl: 'wss://api.xindong.health/ws',
  demoMode: false,
  version: '1.0.0',
  logLevel: 'warn',
}
```

## 🔍 故障排查

### 问题 1：真机调试时提示"不在以下 request 合法域名列表中"

**解决方案：**
1. 确认已勾选"不校验合法域名"
2. 重新扫码进入真机调试
3. 如果还是不行，使用演示模式

### 问题 2：API 请求返回 401 未登录

**解决方案：**
1. 使用演示模式登录
2. 或者实现真实的微信登录

### 问题 3：手机无法访问电脑的后端服务

**检查清单：**
- [ ] 手机和电脑在同一个 Wi-Fi 网络
- [ ] 后端服务监听 `0.0.0.0`（不是 `127.0.0.1`）
- [ ] 防火墙允许 8080 端口
- [ ] IP 地址正确（`192.168.1.102`）

**测试方法：**
```bash
# 在手机浏览器中访问
http://192.168.1.102:8080

# 应该看到：欢迎使用RuoYi后台管理框架
```

## 📱 完整的真机调试流程

### 1. 准备工作

```bash
# 确认后端服务运行在 0.0.0.0
ps aux | grep ruoyi-admin.jar

# 确认可以通过 IP 访问
curl http://192.168.1.102:8080/

# 重新构建小程序
cd mini-program
npm run build:weapp
```

### 2. 微信开发者工具设置

1. 打开项目：`mini-program/dist`
2. 点击"详情" → "本地设置"
3. 勾选 ✅ "不校验合法域名..."
4. 点击"真机调试"
5. 勾选 ✅ "不校验合法域名"
6. 用手机扫码

### 3. 测试小程序

1. **登录**
   - 点击"🔧 厂家演示模式"

2. **查看首页**
   - 应该显示数据（演示数据或真实数据）

3. **测试蓝牙**
   - 点击"开始训练"
   - 搜索 GY-BLE25T 设备
   - 连接设备

4. **查看日志**
   - 在微信开发者工具的控制台查看日志
   - 确认 API 调用成功
